# Homepage Redesign Task

## Goal

Replace the current scaffold-like home screen with a medical-center billing dashboard aligned with `.ai/product-spec.md` and inspired by the dark card-based UI direction shown in the reference image.

## Scope

- Use Tailwind CSS for the page styling.
- Preserve patient search/autofill behavior.
- Reorganize the page around:
  - patient info
  - shift selection
  - operation tabs
  - operation-specific billing forms
  - report/print action
- Prefer reusable sections and components over one-off markup.

## Acceptance notes

- The screen should feel like an intentional desktop dashboard, not a starter template.
- The UI should remain practical for non-technical operators.
- The layout can differ from the wireframe as long as the workflow remains simple.
