# Task Center

Use this folder as the task system for the repo knowledge center.

## Folders

- `inbox/`
  - raw notes from the user in any language
- `triage/`
  - older task notes that still need status review or consolidation
- `planned/`
  - approved structured tasks that are not yet in progress
- `active/`
  - the task files currently being worked on
- `blocked/`
  - structured tasks waiting on a decision, dependency, or external change
- `done/`
  - completed structured tasks
- `templates/`
  - reusable task-file templates

## Workflow

1. Capture raw requests in `inbox/raw-user-tasks.md`.
2. Combine related raw requests into one structured task file when practical.
3. Add or update the task entry in `registry.md`.
4. Move the task file between status folders as the work progresses.

## Tracking

- Canonical status meanings: `task-statuses.md`
- Current task list: `registry.md`
- Recommended task shape: `templates/task-template.md`
